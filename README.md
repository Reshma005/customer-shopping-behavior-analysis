# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI.  
The goal is to understand purchasing patterns, customer segments, and revenue trends.

The project demonstrates an end-to-end data analytics workflow including data cleaning, SQL analysis, and dashboard visualization.

## Tools Used
- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Power BI
- GitHub

## Dataset
The dataset contains customer shopping information such as:

- Customer ID
- Age
- Gender
- Category
- Item Purchased
- Purchase Amount
- Review Rating
- Subscription Status
- Shipping Type
- Discount Applied
- Previous Purchases

## Project Workflow

### 1. Data Cleaning (Python)
- Loaded dataset using Pandas
- Explored dataset structure
- Cleaned and prepared data for analysis

### 2. Data Analysis (SQL)
- Loaded dataset into PostgreSQL
- Wrote SQL queries to analyze customer behavior
- Generated insights like top products, revenue trends, and customer segments

### 3. Data Visualization (Power BI)
- Built interactive dashboards
- Visualized sales trends, customer demographics, and purchasing behavior

## Project Files

- `customer_shopping_behavior.csv` → Dataset
- `customer_behavior_dashboard.sql` → SQL queries
- `customer_behavior_dashboard.pbix` → Power BI dashboard
- `customer_behavior_analysis.ipynb` → Python data analysis
